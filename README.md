# VBA Accounts Payable Receivable Automation
A simple Excel VBA automation tool that processes **invoices (AR)** and creates **journal entries (AP)** automatically.  
This demo project showcases real‑world finance automation using Excel macros.

---

## Features
### **Invoice Processing (AR)**
- Auto‑generates invoices
- Incremental invoice numbering
- Updates the Invoice Log sheet
- Automatically saves invoices to the user's Documents folder

### **Payment Run Processing (AP)**
- Creates upload‑ready journal entries
- Supports multiple vendor payments
- Follows double‑entry accounting logic

---

## How to Use

1. Open the file:  
   **Visual.Basic.Application.Project.xlsm**
2. Enable macros  
3. Press **ALT + F11** to view/edit the VBA code  
4. Use buttons on:
   - **Invoice Processing** sheet  
   - **Payment Run Processing** sheet
